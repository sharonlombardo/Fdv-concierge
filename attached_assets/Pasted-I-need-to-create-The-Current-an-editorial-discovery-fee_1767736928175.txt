Card Component Design:
For Feature/Large cards:

Full-width or half-width image
Title overlaid on image (white text, subtle shadow)
PinButton top-right
Subtle gradient at bottom for text readability

For Style cards:

Product/outfit image
Title below image
PinButton top-right

For Scene cards:

Image focused (minimal or no text)
PinButton top-right
Caption on hover or below

For Quote cards:

Warm neutral background (#f5f1ed)
Elegant serif text centered
PinButton top-right

Each card's PinButton should use:

itemType: matches card type ("feature", "style", "scene", "quote")
itemId: the content id ("current-1", "current-2", etc.)
sourceContext: "current_feed"
aestheticTags: the tags array from the content
itemData: { title, subtitle, imageUrl }

Add route to the app router:
tsx<Route path="/current" component={CurrentFeed} />
Add navigation link in the hamburger menu: "The Current" (place it prominently, not at bottom with debug)

Generous whitespace
Cards have subtle shadows on hover
