## How to Update Images

1. Navigate to `/image-control` in the app
2. Browse or search for the slot you want to update
3. Click the slot to open the editor
4. Upload a new image or paste a URL
5. Preview the change before saving
6. Save to apply the update

### Image Requirements

- **16:9 images**: Hero banners, covers (recommended: 1600x900px minimum)
- **4:5 images**: Portrait/tall images (recommended: 800x1000px minimum)
- **1:1 images**: Square grid tiles (recommended: 600x600px minimum)

### Best Practices

- Use high-quality images with good lighting
- Ensure images are appropriately cropped for their aspect ratio
- Test images in both light and dark modes
- Keep file sizes under 2MB for optimal performance
- Use JPEG for photos, PNG for graphics with transparency
